# BIOST2154-Fall2025-Final
This is a repository storing the Final project files of BIOST 2154: Statistical Methods for Omics Data, taken in Fall 2025. This contains the analysis pipeline and supporting code for a single-cell chromatin accessibility study of mid-gestational human cortical development.

## Project Description
Using single-cell ATAC-seq (scATAC-seq), this project investigates how the regulatory landscape of the developing cerebral cortex is remodeled across post-conception weeks (PCW) 16-24 of neurogenesis, a time window that is implicated in the origins of neurodevelopmental disorders.

## Software
All analyses used R for computation.

## Data Availability
Data used for the analyses presented in this work are available at: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE162170.
